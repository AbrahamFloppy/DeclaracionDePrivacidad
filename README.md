# DeclaracionDePrivacidad
Política de Privacidad de [Nombre de la Aplicación]
1. Datos que recopilamos
Nuestra aplicación recopila los siguientes tipos de datos de los usuarios:

Información básica de perfil: nombre, correo electrónico, foto de perfil y género (según lo autorizado por Facebook).

Datos de uso: interacciones dentro de la app, frecuencia de uso, preferencias seleccionadas.

Contenido compartido: publicaciones, comentarios y mensajes enviados a través de la aplicación.

2. Uso de los datos
Utilizamos los datos recopilados para:

Personalizar la experiencia del usuario dentro de la aplicación.

Mejorar el rendimiento y funcionalidad de la app.

Enviar notificaciones relevantes sobre actualizaciones, promociones o cambios en el servicio.

Analizar patrones de uso para mejorar nuestros servicios.

3. Compartición de datos
No compartimos tus datos personales con terceros, excepto en los siguientes casos:

Proveedores de servicios que nos ayudan a operar la aplicación (por ejemplo, alojamiento web o análisis de datos), bajo estrictos acuerdos de confidencialidad.

Requisitos legales: si la ley lo exige, podemos divulgar información a autoridades competentes.

4. Seguridad de la información
Tomamos medidas técnicas y organizativas para proteger tus datos:

Cifrado de datos sensibles en tránsito y en reposo.

Acceso restringido a la información personal solo a personal autorizado.

Monitoreo continuo para detectar y prevenir accesos no autorizados.
